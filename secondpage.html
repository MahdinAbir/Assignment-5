Question-1: What are the different ways to select an element in the DOM?


There are many ways to select an element in the DOM. Generally getElementsById() is one of the most commonly used to target an ID and select it. There are also other ways to select an element in the DOM . For example :
1.getElementbyClassName()
2. getElementsbyTagname()
3.querySelector()
4.querySelectorAll()...

Question-2: What is the difference between innerHTML, innerText, and textContent ?


These three terms are different and they are used in different condtions. For example,
innerHtml - it selects all the data inside a tag, including other html tags, text content, and everything inside that DOM.
innerText- it only selects the text content inside a particular tag or DOM.
textContent- it is almost similar to innerText but it can also select hidden contents inside the particular tag.

Question-3: What is event delegation in the DOM?


Event delegation is actually using addeventlistener on a parent node so that we can dynamically use eventlistener on the children node. If we use event delegation we can easily handle the situation when there are many child element and we have to use eventlistener on each of them. Delegation simply makes this work simple.

Question-4: What is event bubbling in the DOM?


Event Bubbling is such a event when we specifically target any element of html, the function automatically gets to the root of the targetted element, then it starts to go reverse to each of the elements parent node and so on until it reaches to main document or main parent tag of the html file.
